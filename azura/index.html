Azura

Improving Azura’s Gamification To Increase User Activity And User Engagement

Industry

Game

Timeline

2023

Platform

Website

Scope

UX Design, UI Design, Animation

About

Azura is a community platform for gamers based in France.

Gamers can connect, communicate, and share each others. Azura allow gamers to build their popularity by do streaming and build a professional profile.

Background

Azura want to improve gamification on the platform to increase user activity and user engagement.

As Azura is a social media platform for gamers, they need to give gamification touch as the behaviour of gamers. This things can make users more enjoy using the platform, also to increase their activity and engagement.

Challenge

How do we utilize the gamification features to make it as a source of revenue.

After increase user activity and user engagement through the gamification, we should maximize the potential of the gamification feature as a source of revenue for Azura.

Strategy

Badge Collections

Badge indicates that the user has achieve an achievement. We provide lots of missions to get badges, so users compete to complete missions to get cool badges. This will increase user activity on the platform.

Frame Collections

We present lots of frame collections. User can use frame to make their profile picture more cool. User can get frame by doing mission or buying loot box. This will increase user activity and revenue.

Items Inventory

As we want to improve gamification, so we give a gamification touch in some parts on the platform, one of them is inventory. Inventory contains items that user have like frame, badge, spells, potions, etc.

Sell Chest Loot Box

As we want to maximize the potential of the gamification touch for more revenue, one of the idea is selling chest loot box. User can buy chest loot box to get items like frame, spells, potions.

Leaderboard

There are several categories on the leaderboard, such as most followers, most streaming viewers, overall ranking. This leaderboard aims to encourage users to compete to enter the list, so that it will increase user activity and user engagement.
